# Buscador de Autos

[Buscador de Autos](https://adolfodelarosades.github.io/JS-Proyecto-11-BuscadorAutos/)

El proyecto Buscador de Autos nos permitirá buscar un auto por:

* Marca
* Año
* Precio Mínimo
* Precio Máximo
* No. de Puertas
* Transmisión
* Color

Lo que se utiliza:

* Higher Order Functions
